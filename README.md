# Fetch Recipes

This is a SwiftUI-based recipe app that allows users to browse and explore various dessert recipes using the API https://themealdb.com/api.php.

## Requirements

- Xcode 12.0+
- iOS 14.0+
- Swift 5.3+

## Setup

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Shontinique-Uqdah/fetch_rewards.git
   ```

2. **Open the project in Xcode:**

   ```bash
   cd fetch_rewards
   open Fetch_RecipesApp.swift
   ```

## Usage

### Running the App

1. **Build and run the app:**

   Select your target device (Simulator or a physical device) and press `Cmd + R` to build and run the app.

2. **Explore Recipes:**

   - The app will display a list of recipes.
   - Tap on a recipe to view its detailed information, including ingredients and instructions.

### Running Tests

1. **Unit Tests:**

   Select the `FetchRecipesTests` target and press `Cmd + U` to run the unit tests.

2. **UI Tests:**

   Select the `FetchRecipesUITests` target and press `Cmd + U` to run the UI tests.

## Expected Output

Upon running the app, you should see a list of recipes with images and titles. Selecting a recipe will navigate to a detailed view displaying the recipe's image, ingredients, and instructions.
